## From autocomplete to agents

A year ago AI in my editor meant Tab-completion. Today I run full agentic loops where a model reads the repo, plans a change, edits files, runs tests, and opens a PR — and I review the diff.

## Claude Code

Claude Code lives in the terminal. It has direct filesystem access and runs shell commands. The things that changed my workflow:

- **Hooks** — shell commands that fire before/after tool calls. I use one that runs ESLint after every edit.
- **Skills** — markdown files that teach the model project-specific patterns (caveman mode, commit style, etc.).
- **MCP servers** — expose external tools (Playwright, Serena LSP) as native capabilities.

## opencode

For quick back-and-forth I reach for opencode. It's lighter, feels more like a chat that happens to know your files.

## What I actually delegate

- Mechanical refactors (rename, move, extract)
- Boilerplate generation (content schemas, CRUD routes)
- First-pass code review comments

I still write architecture, design decisions, and anything involving business logic from scratch. The model is a fast junior, not a tech lead.
